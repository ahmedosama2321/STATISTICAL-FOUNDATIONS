1. Descriptive Statistics
- Purpose: Summarize and describe data.
- Tools:
- Measures of central tendency: mean, median, mode
- Measures of dispersion: variance, standard deviation, range
- Data visualization: histograms, boxplots, scatterplots
2. Probability Theory
- Purpose: Model uncertainty and randomness.
- Concepts:
- Probability rules (addition, multiplication)
- Conditional probability & Bayes’ theorem
- Random variables (discrete vs continuous)
- Probability distributions (Normal, Binomial, Poisson)
3. Inferential Statistics
- Purpose: Draw conclusions about populations from samples.
- Methods:
- Estimation (point estimates, confidence intervals)
- Hypothesis testing (t-tests, chi-square, ANOVA)
- p-values and statistical significance
4. Regression & Correlation
- Purpose: Explore relationships between variables.
- Techniques:
- Correlation coefficients
- Simple & multiple linear regression
- Logistic regression for categorical outcomes
5. Foundations of Sampling
- Purpose: Ensure representativeness and reliability.
- Types:
- Random sampling
- Stratified sampling
- Cluster sampling
- Key idea: Larger, random samples reduce bias and improve accuracy.
6. Modern Extensions
- Bayesian statistics: Updating beliefs with evidence
- Non-parametric methods: When assumptions don’t hold
- Resampling techniques: Bootstrapping, permutation tests

🧭 How to Approach Learning
- Start with descriptive statistics to build intuition.
- Move into probability theory for the mathematical backbone.
- Practice inferential methods with real datasets.
- Apply regression models to understand relationships.
- Explore Bayesian and modern methods once the basics are solid.
